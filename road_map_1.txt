BARE-METAL MINI PROJECT SET (NO RTOS)

================================
NON-REAL-TIME MINI PROJECTS
================================

1. GPIO Control + Debounced Button
   - LED ON/OFF using button
   - Proper software debouncing
   - No delay-based debouncing

2. UART Command-Line Interface
   - Simple CLI over UART
   - Commands: LED ON/OFF, STATUS
   - Blocking TX, non-blocking RX

3. Internal Flash Parameter Storage
   - Store configuration data
   - Read after reset
   - Handle erase/write safely

4. I2C Sensor Reader (Polling)
   - Read temperature/light sensor
   - Handle NACK / bus errors
   - No interrupts required

5. RTC Timekeeping (Basic)
   - Configure RTC
   - Read current date/time
   - Backup time across reset

================================
REAL-TIME MINI PROJECTS
================================

6. Timer-Based LED Scheduler
   - Hardware timer interrupt every 10ms
   - Precise LED toggling
   - No delays anywhere

7. ADC Sampling With Timer Trigger
   - Timer triggers ADC periodically
   - ADC ISR stores samples
   - Fixed sampling interval

8. Interrupt-Driven UART (Ring Buffer)
   - RX interrupt fills circular buffer
   - Main loop parses input
   - No busy waiting

9. SPI Driver (Interrupt or DMA-Based)
   - SPI transmit/receive without blocking
   - Handle chip-select correctly
   - Data integrity verification

10. RTC Alarm Interrupt
    - Configure RTC alarm
    - Wake system on alarm event
    - Event-driven logic

================================
INTEGRATION MINI PROJECTS
================================

11. SD Card Interface (SPI Mode)
    - Initialize SD card
    - Read/write blocks
    - Handle timeouts and errors

12. Multi-Task Superloop (Manual Scheduler)
    - Sensor task
    - Logger task (SD card)
    - Communication task
    - No RTOS, non-blocking design

13. Mini Data Logger (Bare-Metal)
    - Timer-based periodic sampling
    - Timestamp data using RTC
    - Store data to SD card
    - UART command to dump status

================================
LOW-POWER PRACTICE
================================

14. Low-Power RTC Wakeup
    - Enter sleep (WFI)
    - RTC alarm wakes MCU
    - Measure sleep duration

================================
RULES FOR ALL PROJECTS
================================

- No RTOS
- No delay() for timing
- Use hardware timers
- ISRs must be minimal
- Main loop must be non-blocking
